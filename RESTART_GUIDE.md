このアプリ（Care Shift AI）の開発を再開したり、Web公開版を更新したりする手順書です。
1年後でも、この手順通りにやれば大丈夫です。

## 1. プロジェクトを開く
このエディター（Antigravity）で、「フォルダを開く」から以下のフォルダを選択してください。
`c:\Users\AIshi\.gemini\antigravity\scratch\new_project\mystic-shifts`

## 2. 開発サーバーの起動（プレビュー用）
自分のPC上で動作確認をしたい場合に使います。

1. ターミナルで以下のコマンドを実行：
   ```bash
   npm run dev
   ```
2. 出てきたURL（`http://localhost:5173/` など）を開く。

## 3. Web公開版の更新（本番用）
修正が終わって、インターネット上のアプリ（`https://care-shift-ai.vercel.app`）を更新したい場合に使います。

1. ターミナルで以下のコマンドを実行：
   ```bash
   npx -y vercel --prod
   ```
2. 処理が終わるのを待つだけで完了です。
